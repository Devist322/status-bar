# Статус-бар 1000/1000

Простое веб-приложение: в верхней части отображается текущее значение и максимум (`1000/1000`), ниже — красивый статус-бар. При нажатии на кнопки `+` и `−` открывается модальное окно «Введите значение», после чего статус-бар изменяется.

## Логика

- **Максимум**: 1000.
- **Текущее значение**: не может быть меньше 0 и больше 1000.
- **Кнопка `+`**: прибавляет введённое целое неотрицательное число к текущему значению, результат ограничивается диапазоном \[0; 1000\].
- **Кнопка `−`**: вычитает введённое целое неотрицательное число из текущего значения, результат ограничивается тем же диапазоном.
- Над баром всегда показывается `текущее/максимум`.

## Запуск

1. Установите Node.js (если ещё не установлен).
2. В корне проекта выполните:

   ```bash
   npm install
   ```

3. Запустите простой сервер:

   ```bash
   npm start
   ```

4. Откройте страницу в браузере:

   ```bash
   http://localhost:3000
   ```

   Если `serve` откроет другой порт, смотрите его в выводе команды.
